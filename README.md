# KelvinFlatship

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19109463.svg)](https://doi.org/10.5281/zenodo.19109463)

Julia repository for the paper: "Linear Kelvin Wave predictions as $z\to 0$"

All data figures can be reproduced using the scripts in the make_figures folder. The tables can be reproduced using the flatship_table function in src/flatship.jl although your times will vary depending on the machine you use. 

If you use this work, please cite the paper. The repository itself can be cited via the DOI badge above if needed.
